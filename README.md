# PayNet Website

Official landing page for PayNet protocol.

**Live at:** https://Chargexpay.com

## Features

- Modern React + Vite setup
- Tailwind CSS styling
- shadcn/ui components
- Multi-page routing with Wouter
- Live payment demo

## Pages

- `/` - Homepage with features and pricing
- `/documentation` - Tools and products overview
- `/quickstart` - Integration guide
- `/demo` - Live x402 payment demo
- `/dashboard` - Analytics dashboard

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Express backend
- PostgreSQL database
